Redis 6.0 源码注释
-----------------

本项目是注释版的 Redis 6.0 源码，原始代码：https://github.com/redis/redis/tree/6.0.16

项目初衷是为了学习 Redis 6.0 新的特性以及进一步了解其内部实现而制作的


阅读顺序 & 源码文件简介
-----------------

| 序号 | 文件 | 简介 |
| ------------- | ------------- | ------------- |
| 1  | `sds.h`、``sds.c` | 简单动态字符串数据结构的实现 |
| 2  | `adlist.h`、``adlist.c` | 双向链表数据结构的实现 |
